# BRUCE

**BRUCE** (**B**oundary **R**ecognition **U**nder **C**rowd **E**licitation) is a web platform for crowd-sourcing laypeople's intuition about text structure through large-scale segmentation experiments.

## Quick Start

Requires **Node.js v20.6+**.

Clone the repository and `npm install` all dependencies. Create a `.env` file based on `.env.example` and edit the variables as needed. Then start the production server:

```bash
npm run build
node --env-file=.env build
```
